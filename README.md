# Grok.cr

Crystal bindings for [Grok](https://github.com/GrokImageCompression/grok) 10.0.8

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     grok:
       github: phenopolis/grok.cr
   ```

2. Run `shards install`
